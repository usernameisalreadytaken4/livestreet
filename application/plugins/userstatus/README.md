User Status plugin for [LiveStreet CMS](http://livestreetcms.com/ "LiveStreet CMS")
===================================================================================

Требования
----------
LiveStreet CMS версии **2.0.0** или более поздней.


Описание
--------
Добавляет простой текстовый статус в профиль пользователя.
