--- Валидаторы ---

В админку добавлены два новых специальных валидатора: Array и Enum


1. Валидатор типа Array предназначен для валидации массива, предоставляя возможности для проверки мин. и макс. количества элементов в массиве, а также для валидации каждого значения массива по отдельному типу валидатора


2. Валидатор типа Enum предназначен только для строковых значений, которые могут быть заданы массивом. Для чисел валидатор не актуален т.к. при указании небольшого диапазона (не более 500) чисел для валидатора типа "число", автоматически подключается селект с возможностью выбора чисел в селекте.

Для данного валидатора нет возможности назначить множественный выбор. Для возможности множественного выбора элементов следует использовать тип валидатора "массив" (подробнее см. в примерах конфига в файле config/config_scheme_sandbox.php
